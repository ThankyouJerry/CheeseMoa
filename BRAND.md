# 치즈모아 브랜드

## 이름

**치즈모아** / **CheeseMoa**

- `치즈`: 치지직 방송을 친근하게 연상시키는 한국어 중심 표현
- `모아`: 여러 방송을 한 화면에 모아 본다는 제품의 핵심 행동
- 한 줄 설명: **보고 싶은 방송, 한곳에.**
- 제품 설명: **외부 멀티뷰 서버 없이 브라우저에서 실행되는 로컬 멀티뷰**

제품 화면과 한국어 문서에서는 `치즈모아`를 우선 사용합니다. 확장 프로그램 이름,
폴더명, 코드 식별자에는 `CheeseMoa`를 사용합니다.

## 시각 아이디어

치즈 조각 안에 여러 개의 시청 창을 배치해 `방송을 모아 본다`는 기능을 표현합니다.
코랄색 재생 창은 현재 집중 중인 방송을 뜻합니다. 기존 프로젝트의 이름, 로고,
클래퍼보드 및 외부 호스팅 자산은 사용하지 않습니다.

## 색상

| 토큰 | 색상 | 용도 |
| --- | --- | --- |
| Deep Navy | `#173B57` | 로고 바탕, 주요 텍스트 |
| Cheese | `#FFC83D` | 브랜드 주색, 선택 상태 |
| Coral | `#FF6B4A` | 집중 방송, 주요 행동 |
| Cream | `#FFF8E5` | 기본 배경 |
| Powder | `#DDECF2` | 보조 배경과 구분선 |
| Slate | `#617282` | 보조 텍스트 |

## 문체

- 짧고 자연스러운 한국어를 사용합니다.
- 기술 용어보다 사용자가 수행할 행동을 먼저 설명합니다.
- `방송 추가`, `한곳에 모아보기`, `집중해서 보기`처럼 결과가 드러나는 표현을 씁니다.

## 상표 표기

치즈모아는 CHZZK 또는 NAVER의 공식 서비스가 아닌 독립 프로젝트입니다. 공식 명칭과
로고는 기능 호환성을 설명하는 데 필요한 범위에서만 사용합니다.
