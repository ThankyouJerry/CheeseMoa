# 치즈모아 CheeseMoa 개인정보 처리방침

**최종 업데이트: 2026년 7월 11일**

치즈모아는 여러 CHZZK 라이브와 다시보기를 한 화면에 배치하는 Chrome 확장 프로그램입니다.
개인정보를 수집하거나 외부 서버로 전송하지 않습니다.

## 기기에 저장하는 정보

사용자가 직접 추가한 CHZZK 라이브·다시보기 URL만 `chrome.storage.local`에 저장합니다. 이
데이터는 현재 Chrome 프로필에만 남으며 치즈모아 개발자 또는 제3자에게 전송되지 않습니다.

## 사용하는 권한

| 권한 | 목적 |
| --- | --- |
| `storage` | 사용자가 추가한 방송 URL 목록을 현재 브라우저에 저장합니다. |
| `https://chzzk.naver.com/*` | 사용자가 추가한 CHZZK 라이브와 다시보기를 멀티뷰 프레임에 불러옵니다. |

치즈모아는 CHZZK 로그인 쿠키를 읽거나 복사하지 않습니다. 로그인과 쿠키는 Chrome과 CHZZK가
직접 관리하며, 치즈모아는 쿠키 값을 저장하거나 외부로 전송하지 않습니다.

## 수집하거나 전송하지 않는 정보

- 네이버 또는 CHZZK 로그인 정보와 쿠키
- 전체 방문 기록, 검색 기록과 시청 기록
- 사용자 이름, 이메일 주소와 연락처
- 사용 통계, 분석 이벤트와 광고 식별자

치즈모아에는 외부 API, 분석 SDK, 광고 네트워크 또는 원격 실행 코드가 없습니다.

## 데이터 삭제

방송 목록에서 개별 URL을 삭제할 수 있습니다. 확장 프로그램을 제거하면 Chrome이 해당 확장
프로그램의 로컬 데이터를 삭제합니다.

## 변경 사항과 문의

정책 변경은 이 문서의 최종 업데이트 날짜와 GitHub 변경 기록에 반영합니다. 문의와 오류 제보는
[GitHub Issues](https://github.com/ThankyouJerry/CheeseMoa/issues)를 이용해 주세요.

## Chrome Web Store Limited Use

치즈모아의 사용자 데이터 사용은 Chrome Web Store 사용자 데이터 정책의 Limited Use 요건을
준수합니다. 사용자 데이터는 확장 프로그램의 단일 목적인 CHZZK 멀티뷰 기능을 제공하는 데만
사용되며, 광고·신용 평가·제3자 판매 또는 사람의 열람에 사용되지 않습니다.
