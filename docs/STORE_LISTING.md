## 상세 설명

**보고 싶은 CHZZK 방송을 한곳에 모아보세요.**

치즈모아는 여러 CHZZK 라이브와 다시보기를 하나의 Chrome 탭에 배치하는 로컬 멀티뷰 확장
프로그램입니다. 별도 회원가입이나 외부 멀티뷰 서버 없이 브라우저에서 바로 실행됩니다.

**주요 기능**

- CHZZK 라이브와 다시보기 URL 여러 개 동시 입력
- 방송 수에 맞춘 1~3열 자동 화면 배치
- 자동, 1열, 2열, 3열 수동 레이아웃
- 개별 방송 새로고침, 제거와 전체화면
- 동일 링크 중복 방지
- Chrome에 로그인된 기존 CHZZK 세션 사용
- 방송 목록을 현재 Chrome 프로필에만 저장

**사용 방법**

1. 도구 모음에서 치즈모아 아이콘을 누릅니다.
2. CHZZK 라이브 또는 다시보기 URL을 한 줄에 하나씩 입력합니다.
3. `방송 모아보기`를 눌러 멀티뷰를 엽니다.
4. 상단 버튼으로 원하는 화면 배치를 선택합니다.

**개인정보 보호**

치즈모아에는 분석 서버, 광고 SDK 또는 사용자 추적 코드가 없습니다. 입력한 CHZZK URL은
현재 Chrome 프로필에만 저장되며 외부로 전송되지 않습니다. 로그인 쿠키를 읽거나 복사하지
않습니다.

치즈모아는 CHZZK 또는 NAVER의 공식 서비스가 아닌 독립 오픈소스 프로젝트입니다.

## 단일 목적 설명

치즈모아의 단일 목적은 사용자가 직접 입력한 여러 CHZZK 라이브와 다시보기를 하나의 Chrome
탭에 멀티뷰로 표시하고 관리하는 것입니다.

## 권한 정당화

### Storage

사용자가 직접 추가한 CHZZK 방송 URL 목록을 현재 Chrome 프로필에 저장하고 다음 실행에서
복원하는 데 사용합니다. 데이터는 외부로 전송하지 않습니다.

### Host permission: `https://chzzk.naver.com/*`

사용자가 입력한 CHZZK 라이브와 다시보기 페이지를 확장 프로그램 내 멀티뷰 프레임으로
불러오는 핵심 기능에만 사용합니다. 페이지 내용을 수집하거나 외부로 전송하지 않습니다.
